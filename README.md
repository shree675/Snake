# Snake

## About
<p>This is a simple recreation of the classic arcade game-Snake using Python.</p>
<p>Control the snake using up :arrow_up:, down :arrow_down:, left :arrow_left: and right :arrow_right: arrow keys.</p>

## Libraries used
* pygame
* time
* random
